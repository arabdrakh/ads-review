🗺️ Graph Traversals & Shortest Path

Week 9: BFS, DFS, and Dijkstra's Algorithm

Comparison

Algorithm Data Structure Order Use Case
DFS Stack Deep first, then backtrack Maze solving, cycle detection
BFS Queue Level by level Shortest path (unweighted)
Dijkstra Priority Queue Lowest distance first Shortest path (weighted)

Depth-First Search (DFS)

Go as deep as possible, then backtrack.

Uses Stack (or recursion) — LIFO order 
void dfs(int start, boolean[] visited) {
    visited[start] = true;
    System.out.print(start + " ");
    
    for (int neighbor : adj[start]) {
        if (!visited[neighbor]) {
            dfs(neighbor, visited);
        }
    }
}

// Iterative with Stack
void dfsIterative(int start) {
    boolean[] visited = new boolean[V];
    Stack<Integer> stack = new Stack<>();
    stack.push(start);
    
    while (!stack.isEmpty()) {
        int v = stack.pop();
        if (!visited[v]) {
            visited[v] = true;
            System.out.print(v + " ");
            
            for (int neighbor : adj[v]) {
                if (!visited[neighbor]) {
                    stack.push(neighbor);
                }
            }
        }
    }
}

Breadth-First Search (BFS)

Visit all neighbors before going deeper.

Uses Queue — FIFO order 
void bfs(int start) {
    boolean[] visited = new boolean[V];
    Queue<Integer> queue = new LinkedList<>();
    
    visited[start] = true;
    queue.add(start);
    
    while (!queue.isEmpty()) {
        int v = queue.poll();
        System.out.print(v + " ");
        
        for (int neighbor : adj[v]) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.add(neighbor);
            }
        }
    }
}
💡 BFS finds the shortest path in unweighted graphs because it explores level by level.

DFS vs BFS Example

Graph:
    0
   /|\
  1 2 3
 /|   |
5 6   4
    
Starting from 0:
DFS: 0 → 1 → 5 → 6 → 2 → 3 → 4 (goes deep first)
BFS: 0 → 1 → 2 → 3 → 5 → 6 → 4 (level by level)

Dijkstra's Algorithm

Finds shortest path from source to all vertices in a weighted graph with non-negative edges.

⚠️ Only works with non-negative edge weights!

Algorithm Steps

  1. Set distance to source = 0, all others = ∞
  2. Add source to priority queue
  3. Extract vertex with minimum distance
  4. Update distances to all unvisited neighbors if shorter path found
  5. Repeat until all vertices processed
void dijkstra(int source) {
    int[] dist = new int[V];
    boolean[] visited = new boolean[V];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[source] = 0;
    
    // Priority queue: (distance, vertex)
    PriorityQueue<int[]> pq = new PriorityQueue<>(
        (a, b) -> a[0] - b[0]
    );
    pq.add(new int[]{0, source});
    
    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int u = curr[1];
        
        if (visited[u]) continue;
        visited[u] = true;
        
        for (int[] edge : adjWeighted[u]) {
            int v = edge[0];
            int weight = edge[1];
            
            if (dist[u] + weight < dist[v]) {
                dist[v] = dist[u] + weight;
                pq.add(new int[]{dist[v], v});
            }
        }
    }
}

Complexity

With priority queue (min-heap): O((V + E) log V)
