        <h1>⚡ Dynamic Programming</h1>
        <h2>Week 10: Optimization through Memoization</h2>

        <div class="section">
            <h2>What is Dynamic Programming?</h2>
            <p>DP solves complex problems by breaking them into <strong>overlapping subproblems</strong> and storing results to avoid redundant computation.</p>

            <div class="key-point">
                <strong>DP = Recursion + Memoization</strong><br>
                Store results of subproblems to reuse them later.
            </div>
        </div>

        <div class="section">
            <h2>When to Use DP?</h2>
            <ol>
                <li><strong>Overlapping Subproblems:</strong> Same subproblem solved multiple times</li>
                <li><strong>Optimal Substructure:</strong> Optimal solution uses optimal solutions of subproblems</li>
            </ol>
        </div>

        <div class="section">
            <h2>DP vs Divide & Conquer</h2>
            <table>
                <tr>
                    <th>Aspect</th>
                    <th>Divide & Conquer</th>
                    <th>Dynamic Programming</th>
                </tr>
                <tr>
                    <td>Subproblems</td>
                    <td>Independent</td>
                    <td>Overlapping</td>
                </tr>
                <tr>
                    <td>Approach</td>
                    <td>Top-down</td>
                    <td>Bottom-up (or top-down with memo)</td>
                </tr>
                <tr>
                    <td>Storage</td>
                    <td>No caching needed</td>
                    <td>Stores results (table/memo)</td>
                </tr>
                <tr>
                    <td>Example</td>
                    <td>Merge Sort</td>
                    <td>Fibonacci, Knapsack</td>
                </tr>
            </table>
        </div>

        <div class="section">
            <h2>Example: Fibonacci</h2>

            <h3>❌ Naive Recursion — O(2ⁿ)</h3>
<pre>
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);  // Redundant calls!
}

// fib(5) calls fib(3) twice, fib(2) three times, etc.
</pre>

            <h3>✅ DP with Memoization — O(n)</h3>
<pre>
int[] memo = new int[n + 1];
Arrays.fill(memo, -1);

int fibMemo(int n) {
    if (n <= 1) return n;
    if (memo[n] != -1) return memo[n];  // Already computed!
    
    memo[n] = fibMemo(n-1) + fibMemo(n-2);
    return memo[n];
}
</pre>

            <h3>✅ DP Bottom-Up — O(n)</h3>
<pre>
int fibBottomUp(int n) {
    if (n <= 1) return n;
    
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    
    return dp[n];
}
</pre>

            <h3>✅ Space Optimized — O(1) space</h3>
<pre>
int fibOptimized(int n) {
    if (n <= 1) return n;
    
    int prev2 = 0, prev1 = 1;
    
    for (int i = 2; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    
    return prev1;
}
</pre>
        </div>

        <div class="section">
            <h2>Steps to Solve DP Problems</h2>
            <ol>
                <li><strong>Define subproblems</strong> — What state do we need?</li>
                <li><strong>Write recurrence relation</strong> — How does current depend on previous?</li>
                <li><strong>Identify base cases</strong> — What's the simplest case?</li>
                <li><strong>Decide iteration order</strong> — Fill table in correct order</li>
                <li><strong>Optimize space</strong> (if possible)</li>
            </ol>
        </div>
